## Шарипов Данил РИС-22-1.
# Адаптивная система тестирования студентов-биологов
Веб-система тестирования знаний, предназначенная для студентов биологических направлений.
Система позволяет проводить тестирование по учебным темам, автоматически проверять ответы, сохранять результаты в базе данных и формировать новые тесты для последующего улучшения знаний именно в той теме, где у студента мало знаний.
